# HTML 요소의 구조

HTML 요소는 '**여는 태그 + 내용 + 닫는 태그**'로 구성됩니다
일부 요소는 닫는 태그가 필요 없는 빈 요소로 작성됩니다

<br/>

## 📌 HTML 요소 예제

```html
<!-- 여는 태그 -->
<p>
    <!-- 내용 -->
    이것은 문단입니다.
</p>
<!-- 닫는 태그 -->

<!-- 빈 요소 -->
<img src="image.jpg" alt="이미지 설명" />
```
